# VLAN & Inter-VLAN Routing Lab

##  Description
This project demonstrates how to configure VLANs and enable communication between them using Router-on-a-Stick.

##  Technologies Used
- VLANs
- Trunking
- Router-on-a-Stick
- Cisco Packet Tracer

##  Network Design
- 1 Router (Cisco 2811)
- 1 Switch
- 3 PCs
- 3 VLANs (10, 20, 30)

##  Configuration Highlights
- VLAN creation on switch
- Access port assignment
- Trunk port configuration
- Sub-interfaces on router
- Inter-VLAN routing

##  Result
All devices across different VLANs can communicate successfully using ping.
